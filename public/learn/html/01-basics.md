## HTML이란 무엇인가?

HTML(HyperText Markup Language)은 웹 페이지의 **뼈대**를 만드는 언어입니다.

건물을 지을 때 철근과 콘크리트로 뼈대를 세우듯, 웹 페이지도 HTML로 구조를 먼저 잡습니다. 그 위에 CSS로 디자인(색상, 크기, 레이아웃)을 입히고, JavaScript로 동작(클릭, 애니메이션)을 추가합니다.

- **HTML** = 건물의 뼈대 (구조)
- **CSS** = 건물의 인테리어 (디자인)
- **JavaScript** = 건물의 전기·수도 (동작)

HTML은 프로그래밍 언어가 아닙니다. 브라우저에게 "이 내용은 제목이야", "이건 문단이야"라고 **표시(마크업)**해주는 언어입니다.

## 태그(Tag)란?

HTML은 **태그**라는 특별한 기호로 내용을 감쌉니다.

```html
<태그이름>내용</태그이름>
```

- `<태그이름>` : 여는 태그 (시작)
- `</태그이름>` : 닫는 태그 (끝, 슬래시 `/` 주의)
- 두 태그 사이의 내용이 해당 태그의 역할을 가집니다

예시:
```html
<h1>안녕하세요</h1>   <!-- h1 = 가장 큰 제목 -->
<p>반갑습니다.</p>     <!-- p = 문단(paragraph) -->
```

### 자주 하는 실수: 닫는 태그를 빼먹기

```html
<!-- 잘못된 예 -->
<p>이 문단은 닫히지 않았습니다.
<p>그래서 브라우저가 혼란스러워합니다.

<!-- 올바른 예 -->
<p>이 문단은 제대로 닫혔습니다.</p>
<p>다음 문단도 잘 닫혔습니다.</p>
```

## HTML 파일의 기본 구조

모든 HTML 파일은 아래와 같은 기본 틀을 가집니다. 마치 편지를 쓸 때 "받는 사람:", "본문:", "보내는 사람:" 형식이 있듯이, HTML도 정해진 형식이 있습니다.

```html
<!DOCTYPE html>
<!-- 이 파일이 HTML5 문서임을 브라우저에게 알림 -->

<html lang="ko">
<!-- html: 문서 전체를 감싸는 최상위 태그 -->
<!-- lang="ko": 이 페이지가 한국어임을 알림 (스크린 리더, 검색엔진에 중요) -->

  <head>
    <!-- head: 브라우저에게 전달할 설정 정보 (화면에는 보이지 않음) -->

    <meta charset="UTF-8">
    <!-- charset: 문자 인코딩 방식 지정 -->
    <!-- UTF-8: 한글, 영어, 이모지 등 모든 문자를 표현 가능 -->
    <!-- 이 줄이 없으면 한글이 깨져 보일 수 있음! -->

    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <!-- viewport: 모바일 화면 대응을 위한 설정 -->
    <!-- width=device-width: 화면 너비를 기기 너비에 맞춤 -->
    <!-- initial-scale=1.0: 초기 확대 비율을 1배로 설정 -->
    <!-- 이 줄이 없으면 모바일에서 화면이 매우 작게 보임! -->

    <title>내 첫 번째 웹 페이지</title>
    <!-- title: 브라우저 탭에 표시되는 페이지 제목 -->
    <!-- 검색 결과에도 이 제목이 표시됨 -->

  </head>

  <body>
    <!-- body: 실제로 화면에 표시될 내용이 들어가는 곳 -->

    <h1>안녕하세요!</h1>
    <!-- h1: 페이지의 가장 큰 제목 -->

    <p>제 첫 번째 웹 페이지에 오신 것을 환영합니다.</p>
    <!-- p: 일반 텍스트 문단 -->

  </body>

</html>
```

## 각 부분 자세히 알아보기

### `<!DOCTYPE html>`

이 선언문은 "이 파일은 HTML5 방식으로 작성되었습니다"라고 브라우저에게 알려줍니다.

예전 HTML 버전은 이 선언문이 매우 복잡했습니다:
```html
<!-- HTML4 시절 (복잡했음) -->
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

<!-- HTML5 현재 (간단!) -->
<!DOCTYPE html>
```

항상 파일의 **첫 번째 줄**에 와야 합니다.

### `<head>` — 문서의 메타 정보

head 안의 내용은 화면에 보이지 않고, 브라우저와 검색엔진에게 정보를 전달합니다.

```html
<head>
  <meta charset="UTF-8">           <!-- 문자 인코딩 -->
  <meta name="viewport" content="width=device-width, initial-scale=1.0"> <!-- 모바일 대응 -->
  <title>페이지 제목</title>        <!-- 탭 제목 -->
  <link rel="stylesheet" href="style.css"> <!-- CSS 파일 연결 -->
  <script src="script.js"></script>        <!-- JavaScript 파일 연결 -->
</head>
```

### `<body>` — 실제 화면 내용

body 안에 넣은 모든 내용이 브라우저 화면에 표시됩니다.

## 자주 쓰는 기본 태그들

### 제목 태그 (h1 ~ h6)

```html
<h1>가장 큰 제목 (페이지당 1개 권장)</h1>
<h2>두 번째 제목 (섹션 제목)</h2>
<h3>세 번째 제목 (소제목)</h3>
<h4>네 번째 제목</h4>
<h5>다섯 번째 제목</h5>
<h6>가장 작은 제목</h6>
```

h1부터 h6까지 있으며, 숫자가 작을수록 크고 중요한 제목입니다. 책의 "1장 > 1.1절 > 1.1.1항" 구조와 같습니다.

**주의**: 제목 태그는 글자 크기를 조절하는 용도가 아닙니다! 글자 크기는 CSS로 조절하고, 제목 태그는 문서의 **계층 구조**를 나타내는 데 사용합니다.

### 문단 태그

```html
<p>이것은 하나의 문단입니다. 문단과 문단 사이에는 자동으로 여백이 생깁니다.</p>

<p>이것은 두 번째 문단입니다.</p>
```

### 줄바꿈과 구분선

```html
<p>
  첫 번째 줄<br>  <!-- br: 강제 줄바꿈 (닫는 태그 없음) -->
  두 번째 줄
</p>

<hr>  <!-- hr: 수평 구분선 (닫는 태그 없음) -->

<p>구분선 아래 내용</p>
```

### 텍스트 강조

```html
<p>이 단어는 <strong>매우 중요합니다</strong>.</p>
<!-- strong: 중요한 내용 강조 (굵게, 의미적으로 중요함) -->

<p>이 단어는 <em>강조됩니다</em>.</p>
<!-- em: 감조(emphasis), 기울임꼴로 표시 -->

<p><strong>vs</strong> <b>bold</b>의 차이:</p>
<!-- b: 단순히 굵게 (의미 없음) -->
<!-- strong: 의미적으로 중요함 (스크린 리더가 강조해서 읽음) -->
```

### 목록 태그

```html
<!-- ul: 순서 없는 목록 (Unordered List) - 점(•)으로 표시 -->
<ul>
  <li>사과</li>   <!-- li: 목록 항목 (List Item) -->
  <li>바나나</li>
  <li>오렌지</li>
</ul>

<!-- ol: 순서 있는 목록 (Ordered List) - 숫자로 표시 -->
<ol>
  <li>HTML 배우기</li>
  <li>CSS 배우기</li>
  <li>JavaScript 배우기</li>
</ol>

<!-- 중첩 목록 -->
<ul>
  <li>프론트엔드
    <ul>
      <li>HTML</li>
      <li>CSS</li>
    </ul>
  </li>
  <li>백엔드</li>
</ul>
```

## 속성(Attribute)이란?

태그에 추가 정보를 넣을 때 **속성**을 사용합니다. 속성은 여는 태그 안에 작성합니다.

```html
<태그이름 속성이름="속성값">내용</태그이름>
```

예시:
```html
<a href="https://google.com">구글로 이동</a>
<!-- a: 링크 태그 -->
<!-- href: 이동할 주소 (href = hypertext reference) -->

<img src="photo.jpg" alt="내 사진">
<!-- img: 이미지 태그 (닫는 태그 없음) -->
<!-- src: 이미지 파일 경로 -->
<!-- alt: 이미지를 불러올 수 없을 때 표시할 대체 텍스트 -->
```

## 주석 작성하기

코드에 설명을 달고 싶을 때 주석을 사용합니다. 주석은 화면에 표시되지 않습니다.

```html
<!-- 이것은 주석입니다. 브라우저가 무시합니다. -->

<p>이 문단은 화면에 표시됩니다.</p>

<!--
  여러 줄
  주석도
  가능합니다.
-->
```

## 들여쓰기(Indentation)

HTML 코드는 들여쓰기를 잘 해야 나중에 읽기 편합니다. 일반적으로 자식 태그는 부모 태그보다 2칸 또는 4칸 들여씁니다.

```html
<!-- 들여쓰기가 없는 읽기 어려운 코드 -->
<html><head><title>제목</title></head><body><h1>안녕</h1><p>반가워요</p></body></html>

<!-- 들여쓰기가 잘 된 읽기 쉬운 코드 -->
<html>
  <head>
    <title>제목</title>
  </head>
  <body>
    <h1>안녕</h1>
    <p>반가워요</p>
  </body>
</html>
```

## 실습: 간단한 자기소개 페이지 만들기

```html
<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>김개발의 자기소개</title>  <!-- 탭에 표시될 제목 -->
  </head>
  <body>

    <h1>안녕하세요, 김개발입니다!</h1>
    <!-- 페이지의 메인 제목 -->

    <h2>저에 대해서</h2>
    <!-- 섹션 제목 -->

    <p>
      저는 웹 개발을 배우고 있는 초보 개발자입니다.<br>
      <!-- br로 줄을 바꿈 -->
      HTML부터 차근차근 공부하고 있습니다.
    </p>

    <hr>
    <!-- 시각적 구분선 -->

    <h2>관심 기술</h2>

    <ul>
      <!-- 순서가 중요하지 않은 목록 -->
      <li>HTML</li>
      <li>CSS</li>
      <li>JavaScript</li>
    </ul>

    <h2>학습 계획</h2>

    <ol>
      <!-- 순서가 중요한 목록 -->
      <li>HTML 기초 완성하기</li>
      <li>CSS로 스타일링 배우기</li>
      <li>JavaScript로 동작 추가하기</li>
      <li>나만의 포트폴리오 만들기</li>
    </ol>

    <p>
      <strong>목표:</strong> 6개월 안에 첫 번째 프로젝트 완성!
      <!-- strong으로 중요한 내용 강조 -->
    </p>

  </body>
</html>
```

## 자주 하는 초보 실수 모음

```html
<!-- 실수 1: DOCTYPE 선언 빠뜨리기 -->
<html>  <!-- DOCTYPE이 없으면 브라우저가 호환 모드로 동작할 수 있음 -->

<!-- 실수 2: charset 설정 빠뜨리기 -->
<head>
  <title>제목</title>  <!-- charset 없으면 한글이 깨질 수 있음! -->
</head>

<!-- 실수 3: head와 body 순서 바꾸기 -->
<body>...</body>
<head>...</head>  <!-- head가 반드시 body보다 먼저 와야 함 -->

<!-- 실수 4: 태그를 잘못 중첩하기 -->
<p>안녕 <strong>하세요</p></strong>   <!-- 잘못됨 -->
<p>안녕 <strong>하세요</strong></p>   <!-- 올바름: 나중에 연 태그를 먼저 닫아야 함 -->

<!-- 실수 5: h1을 여러 개 사용하기 -->
<h1>제목 1</h1>
<h1>제목 2</h1>  <!-- h1은 페이지당 하나만 사용하는 것이 SEO에 좋음 -->
```

## 정리

| 태그 | 역할 | 예시 |
|------|------|------|
| `<!DOCTYPE html>` | HTML5 문서 선언 | 항상 첫 줄에 작성 |
| `<html>` | 문서 전체 감싸기 | `lang` 속성으로 언어 지정 |
| `<head>` | 문서 설정 정보 | charset, title 등 |
| `<body>` | 화면에 표시될 내용 | 실제 콘텐츠 |
| `<h1>~<h6>` | 제목 | 계층 구조 표현 |
| `<p>` | 문단 | 일반 텍스트 |
| `<br>` | 줄바꿈 | 닫는 태그 없음 |
| `<hr>` | 구분선 | 닫는 태그 없음 |
| `<strong>` | 중요 강조 | 굵게 표시 |
| `<em>` | 일반 강조 | 기울임 표시 |
| `<ul>/<ol>/<li>` | 목록 | 점/숫자 목록 |
