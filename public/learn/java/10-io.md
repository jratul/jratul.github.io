## 자바 I/O 구조

```
java.io   → 전통적 I/O (스트림 기반, 블로킹)
java.nio  → New I/O (버퍼, 채널, 비블로킹 가능)
java.nio.file (NIO.2) → 파일 시스템 API (Java 7+, 권장)
```

실무에서는 대부분 `java.nio.file`의 `Files`, `Path`를 사용합니다.

## Path와 Files (NIO.2)

```java
import java.nio.file.*;

// 경로 생성
Path path = Path.of("data/hello.txt");
Path absolute = path.toAbsolutePath();

System.out.println(path.getFileName());   // hello.txt
System.out.println(path.getParent());     // data
System.out.println(path.toString());      // data/hello.txt
```

### 파일 읽기

```java
// 전체 텍스트 한 번에
String content = Files.readString(Path.of("hello.txt"));

// 줄 단위 리스트
List<String> lines = Files.readAllLines(Path.of("hello.txt"));
for (String line : lines) {
    System.out.println(line);
}

// 스트림으로 읽기 (대용량 파일에 적합)
try (Stream<String> stream = Files.lines(Path.of("big.txt"))) {
    stream.filter(line -> line.contains("error"))
          .forEach(System.out::println);
}
```

### 파일 쓰기

```java
// 텍스트 쓰기
Files.writeString(Path.of("output.txt"), "Hello, World!\n");

// 줄 목록 쓰기
List<String> lines = List.of("첫 번째 줄", "두 번째 줄");
Files.write(Path.of("output.txt"), lines);

// 이어 쓰기 (append)
Files.writeString(
    Path.of("log.txt"),
    "새 로그\n",
    StandardOpenOption.APPEND,
    StandardOpenOption.CREATE
);
```

### 파일/디렉토리 관리

```java
Path file = Path.of("data/sample.txt");
Path dir  = Path.of("data/backup");

// 존재 여부
Files.exists(file);
Files.isDirectory(dir);
Files.isRegularFile(file);

// 생성
Files.createDirectories(dir);       // 중간 디렉토리도 함께 생성
Files.createFile(file);             // 이미 있으면 예외

// 복사 / 이동
Files.copy(file, Path.of("data/backup/sample.txt"),
    StandardCopyOption.REPLACE_EXISTING);
Files.move(file, Path.of("data/new.txt"));

// 삭제
Files.delete(file);
Files.deleteIfExists(file);  // 없어도 예외 없음

// 파일 크기, 수정시각
long size = Files.size(file);
FileTime modified = Files.getLastModifiedTime(file);
```

## 디렉토리 탐색

```java
// 직접 자식만
try (DirectoryStream<Path> stream = Files.newDirectoryStream(Path.of("src"))) {
    for (Path entry : stream) {
        System.out.println(entry);
    }
}

// 재귀 탐색
try (Stream<Path> walk = Files.walk(Path.of("src"))) {
    walk.filter(Files::isRegularFile)
        .filter(p -> p.toString().endsWith(".java"))
        .forEach(System.out::println);
}

// 패턴 매칭
PathMatcher matcher = FileSystems.getDefault().getPathMatcher("glob:**.java");
try (Stream<Path> walk = Files.walk(Path.of("src"))) {
    walk.filter(matcher::matches)
        .forEach(System.out::println);
}
```

## 전통 I/O: BufferedReader / BufferedWriter

NIO.2로 해결 안 되는 경우나 레거시 코드에서 씁니다.

```java
// 읽기
try (BufferedReader reader = new BufferedReader(
        new FileReader("hello.txt", StandardCharsets.UTF_8))) {
    String line;
    while ((line = reader.readLine()) != null) {
        System.out.println(line);
    }
}

// 쓰기
try (BufferedWriter writer = new BufferedWriter(
        new FileWriter("output.txt", StandardCharsets.UTF_8))) {
    writer.write("첫 번째 줄");
    writer.newLine();
    writer.write("두 번째 줄");
}
```

`try-with-resources`를 사용하면 예외가 발생해도 스트림이 자동으로 닫힙니다.

## 직렬화

객체를 바이트로 변환해 파일이나 네트워크로 전송합니다.

```java
// 직렬화 가능한 클래스
class User implements Serializable {
    private static final long serialVersionUID = 1L;
    private String name;
    private int age;

    // 직렬화에서 제외
    transient private String password;
}

// 저장
try (ObjectOutputStream oos = new ObjectOutputStream(
        new FileOutputStream("user.dat"))) {
    oos.writeObject(new User("Alice", 30));
}

// 불러오기
try (ObjectInputStream ois = new ObjectInputStream(
        new FileInputStream("user.dat"))) {
    User user = (User) ois.readObject();
}
```

실무에서는 Java 직렬화 대신 **JSON**(Jackson, Gson)을 주로 씁니다. Java 직렬화는 보안 취약점과 버전 호환 문제가 있습니다.

## 실전 예제: CSV 파일 처리

```java
record Product(String name, String category, int price) {}

// CSV 파일 읽기
List<Product> products = Files.lines(Path.of("products.csv"))
    .skip(1)  // 헤더 건너뜀
    .map(line -> {
        String[] parts = line.split(",");
        return new Product(parts[0], parts[1], Integer.parseInt(parts[2].trim()));
    })
    .collect(Collectors.toList());

// CSV 파일 쓰기
List<String> lines = new ArrayList<>();
lines.add("name,category,price");  // 헤더
products.stream()
    .map(p -> p.name() + "," + p.category() + "," + p.price())
    .forEach(lines::add);

Files.write(Path.of("output.csv"), lines);
```
